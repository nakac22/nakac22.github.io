🔑 1. APIキーの確実な取得方法（完全無料）

本アプリで使用するGemini APIは、個人の学習・検証用途であれば**完全に無料（料金発生なし）**で利用可能です。

  1. Googleアカウントにログインした状態で Google AI Studio にアクセスします。
  2. 画面左側のメニューにある 「🔑 APIキーを取得」（英語: `Get API key`）を開きます。
  3. 画面右上にある青い 「APIキーを作成」 ボタン（英語: `Create API key`）を押します。
  4. 発行されたAPIキーをコピーし、下の「GEMINI API KEY」欄に貼り付けてください。

🚀 2. アプリの基本的な使い方

  • 環境設定： 上記で取得したAPIキーを入力します。
  • 英文生成： 条件を選び、「🎯 パラグラフ英文を生成する」をクリックします。
  • 要約ライティング： 表示された英文を読み、右側に英語で要約文を入力します。
  • 診断・採点： 「🔍 要約を採点・診断する」をクリックするとAIが診断します。

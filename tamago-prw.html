🔑 1. APIキーの確実な取得方法（完全無料）

本アプリで使用するGemini APIは、個人の学習・検証用途であれば**完全に無料（料金発生なし）**で利用可能です。

  1. Googleアカウントにログインした状態で Google AI Studio にアクセスします。
  2. 画面左上の青いボタン 「Get API key」 をクリックします。
  3. 「Create API key」 ボタンを押し、プロジェクトを選択（または新規作成）してキーを生成します。
  4. 生成された英数字の長い文字列（API Key）をコピーし、下の「Gemini API Key」欄に貼り付けてください。

※入力したキーはあなたのブラウザ（Local Storage）にのみ安全に保存され、外部のサーバーに送信されることはありません。

🚀 2. アプリの基本的な使い方

  • 環境設定： 上記で取得したAPIキーを入力します。
  • 英文生成： 画面左側で「英語レベル」「トピック」「論理構造パターン」を選び、「🎯 パラグラフ英文を生成する」をクリックします。論理的シグナルを含んだ問題が自動生成されます。
  • 要約ライティング： 表示された英文を読み、右側の入力欄に英語で要約文を作成・入力します。
  • 診断・採点： 「🔍 要約を採点・診断する」をクリックすると、AIが「論理骨子の抽出度」などを3つの観点から厳格に採点し、日本語で詳細なフィードバックと模範解答を提供します。

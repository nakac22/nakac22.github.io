🔑 1. APIキーの確実な取得方法（完全無料）

本アプリで使用するGemini APIは、個人の学習・検証用途であれば**完全に無料（料金発生なし）**で利用可能です。

  1. Googleアカウントにログインした状態で Google AI Studio にアクセスします。
  2. 画面左側のメニュー（スマホ・タブレットはメニューアイコン）にある 「🔑 APIキーを取得」（英語: `Get API key`）を開きます。
  3. 画面右上にある青い 「APIキーを作成」 ボタン（英語: `Create API key`）を押します。
  4. プロジェクト選択欄で 「新しいプロジェクトを作成」 を選び、プロジェクト名に半角英字（例: Gemini-Project）と入力して作成します。※日本語名は不可。
  5. 発行されたAPIキー（`AQ...` や `AI...`）をコピーし、下の「Gemini API Key」欄に貼り付けてください。

※VPN（通信暗号化アプリ）をオンにしていると通信エラーになる場合があります。一時的にオフにしてご利用ください。

🚀 2. アプリの基本的な使い方

  • 環境設定： 上記で取得したAPIキーを入力します。モデルは無料枠で動く「Flash系」を選択してください。
  • 英文生成： 画面左側で条件を選び、「🎯 パラグラフ英文を生成する」をクリックします。
  • 要約ライティング： 表示された英文を読み、右側の入力欄に英語で要約文を作成・入力します。
  • 診断・採点： 「🔍 要約を採点・診断する」をクリックすると、AIが論理構造を診断して詳細なフィードバックを提供します。

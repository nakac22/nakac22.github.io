📱 スマホでの見つけ方手順：
  1. リンク先でGoogleアカウントにログインし、規約同意のチェックを入れて進みます。
  2. 画面の左上にある「三本線のマーク（メニュー）」をタップします。
  3. メニューの中に現れる、鍵のマークがついた「Get API key」（または Create API key）をタップします。
  4. 次の画面で、青いボタンの「Create API key」を押すと、長い英数字（AIzaSy...）が発行されるので「Copy」します。（何かをダウンロードする必要はありません）

選択 興味のあるトピック（複数選択可）
